右侧结果区用哪种“看得懂”的方式？

你已经选了：工具箱子页面 + 左右分栏 + 实用版能力。现在定最关键的“阅读体验”。

A

格式化原始文本

像 json.cn 那样，右侧重排后的 JSON 文本，带行号和语法高亮。最接近你的原话：“一眼看懂的 JSON”。

B

树形结构视图

对象/数组可折叠展开，键值一目了然。适合层级复杂的数据，但和“原样 JSON”会有一点差别。

C

双视图切换（推荐）

默认显示格式化原始文本，旁边提供“结构视图”按钮切换到树形。两者都保留，体验最稳。
