JSON 解析器应该放在哪里？

先确认入口和页面归属，再细化交互与需求。

A

工具箱子页面（推荐）

放在 /toolbox/json-parser，保留“工具箱”作为工具集合入口，后续可继续扩展其他小工具。

B

独立顶级页面

直接用 /json-parser，导航上单独出现。更直给，但会和工具箱定位重叠。

C

工具箱内嵌面板

先做工具列表，点击“JSON 解析器”后在同一个页面内展开工具面板，不跳新页面。
